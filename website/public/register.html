🎯 Vulnerabilities

A07: No Password Requirements

Any password is accepted, including:

  • Single character: "a"
  • Common passwords: "password", "123456"
  • Username as password

A04: Weak Password Hashing

Passwords are hashed with MD5 - easily crackable with rainbow tables

A05: SQL Injection

The registration form might be vulnerable to SQL injection
