Analysis Answers

1

Which TCP/IP layer delivers the attack?

Application Layer (Layer 4)
SQL injection payloads are embedded in HTTP requests, which operate at the Application layer. The malicious SQL code travels within HTTP POST/GET parameters through the web protocol stack. Lower layers (Transport, Internet, Network Access) simply carry the HTTP data without understanding its content.
2

Which OWASP category applies?

A05:2025 - Injection
SQL injection is the classic example of an Injection vulnerability (moved from A03 in 2021 to A05 in 2025). It occurs when untrusted data is sent to an interpreter as part of a command or query. The attacker's hostile data can trick the interpreter into executing unintended commands or accessing unauthorized data.
3

What could be controlled in a CPS context?

All Building Systems
With admin access to a Building Management System, an attacker could control: HVAC (cause discomfort/damage), Access Control (unlock doors, enable unauthorized entry), Fire Systems (disable alarms, trigger sprinklers), Elevators (trap occupants), Power (cause outages), and Surveillance (disable monitoring, spy on occupants).
