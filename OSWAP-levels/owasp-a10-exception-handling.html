    try:
        user = auth_service.verify(username, password)
        return user
    except ServiceUnavailableError:
        # VULNERABILITY: Fails open!
        return User(role="admin")  # Grant access anyway

Network Relevance

                    
Attackers can leverage edge cases or error paths to bypass checks, crash services (leading to DoS), or expose internal data affecting network controls. When systems "fail open" during exceptional conditions, authentication and authorization controls can be bypassed, allowing unauthorized network access. Verbose error messages can reveal internal network topology, database structures, and other sensitive configuration details.
